가상 화장

당신의 얼굴에 화장할 필요 없어요. 카메라를 통해 화장된 모습을 미리 확인해 봐요.

화장하러 가기

마음에 드는 제품을 빠르게

화장이 마음에 들으셨나요? 빠르게 원하는 화장품을 구매하러 가요.

구매하러 가기

누구나 이해하는 사이트 가이드라인

당신도 할 수 있다. 카메라 연결!

"카메라 연결" 버튼을 클릭하면 웹 카메라가 활성화되어, 화면에 당신의 얼굴이 실시간으로 출력됩니다.

누구나 할 수 있다. 얼굴에 화장하기!

카메라가 연결되면, 화면에 파운데이션과 립 화장 기능을 조절할 수 있는 버튼이 나타납니다. 각 버튼을 클릭하여 원하는 부위를 화장할 수 있습니다.

마음에 드는 화장 저장하기!

로그인한 사용자만 화장 상태를 저장할 수 있습니다. 먼저 저장할 번호(1~5)를 선택하고, 저장하기 버튼을 클릭하면 선택한 색상과 진하기가 저장됩니다.

저장된 화장을 불러옵시다!

저장한 화장 스타일을 다시 불러오려면 불러오기 버튼을 클릭하세요. 먼저 불러올 번호(1~5)를 선택하고, 해당 번호의 저장된 화장 색상과 진하기를 불러와서 바로 적용할 수 있습니다.
