카드 제목

이것은 카드의 본문 내용입니다. 부트스트랩을 사용하여 쉽게 스타일을 적용할 수 있습니다.

자세히 보기

카메라 연결

"카메라 연결" 버튼을 클릭하면 웹 카메라가 활성화되어, 화면에 당신의 얼굴이 실시간으로 출력됩니다.

화장 시작

카메라가 연결되면, 화면에 파운데이션과 립 화장 기능을 조절할 수 있는 버튼이 나타납니다. 각 버튼을 클릭하여 원하는 부위를 화장할 수 있습니다.
각 화장 부위를 선택하면, 슬라이더가 나타나 색상과 진하기를 자유롭게 조절할 수 있습니다. 원하는 색과 진하기를 선택해 보세요!

화장 저장 (로그인 사용자만 가능)

로그인한 사용자만 화장 상태를 저장할 수 있습니다. 먼저 저장할 번호(1~5)를 선택하고, 그 후 저장하기 버튼을 클릭하면 선택한 색상과 진하기가 저장됩니다.

저장된 화장 불러오기 (로그인 사용자만 가능)

저장한 화장 스타일을 다시 불러오려면 불러오기 버튼을 클릭하세요. 먼저 불러올 번호(1~5)를 선택하고, 해당 번호의 저장된 화장 색상과 진하기를 불러와서 바로 적용할 수 있습니다.

화장 체험 서비스
